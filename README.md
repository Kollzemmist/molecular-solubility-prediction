# Molecular Solubility Prediction

This project applies machine learning to predict the aqueous solubility of chemical compounds using molecular descriptors.

## Objective
To build a regression model that estimates solubility based on physicochemical properties of molecules.

## Methodology
- Data preprocessing and cleaning
- Feature selection using molecular descriptors
- Model training using Linear Regression
- Evaluation using R² score

## Results
The model achieved an R² score of approximately 0.73, indicating good predictive performance for a baseline model.

## Tools Used
- Python
- pandas
- scikit-learn
- matplotlib
